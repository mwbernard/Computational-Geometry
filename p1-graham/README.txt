a1 - Convex Hull

by Martin Bernard and Jeonguk Choi


to run from command line:

"./viewPoints [# of desired points]"


KEYPRESS commands:

use 'o' to initialize horizontal line of points

use 'v' to initialize vertical line of points

use 'u' to initialize point square

use 's' to initialize point star

use 'r' to initialize random points

use 'c' to initialize point circle

use 'h' to re-calculate Hull

use 'q' to quit
