p5 - Motion Planning using Visibility Graph

by Martin Bernard and Jeonguk Choi

to run from command line after making:

“./vgmotion”

KEYPRESS commands:

"s" - start drawing a polygon that will be an obstacle

"e" - press when finished drawing your polygon to add the polygon to the obstacles

	note: for each new polygon you add, you click "s" to start drawing it, and click "e" to finalize it

“v” - compute the visibility graph and displays it
 
“p” - compute the visibility graph if there isn’t one, and compute the shortest path from the start (cyan) to the goal (yellow)

“i” - initialize the program

"q" - quit
