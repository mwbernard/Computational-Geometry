p2 - orthogonal_line_intersections

by Martin Bernard and Jeonguk Choi


to run from command line:

"./viewPoints [# of desired points]"


KEYPRESS commands:

use 'c' to clear the current intersections displayed

use 'i' to change the line segment set, there are many to cycle through

use 'r' to re-calculate the intersections and reset the timeline

use 'q' to quit
