p3 - Mondrian KD Tree

by Martin Bernard and Jeonguk Choi

to run from command line after making:

"./viewPoints [# of desired points]"

we don't print out the whole tree, but the info for the tree, and the time for calculation are printed upon each reset.

KEYPRESS commands:

use ' ' to reset KD tree with new points. There are 4 test cases:

	- random points
	- vertical line
	- horizontal line
	- point cross 

use 'q' to quit
