p6 - Motion Planning with the Occupancy Grid

by Martin Bernard and Jeonguk Choi

to run from command line after making:

“./findpath”

once free space has been calculated, you can choose new start and end
points and click p to recompute path

KEYPRESS commands:

"s" - start drawing a polygon that will be an obstacle

"e" - press when finished drawing your polygon to add the polygon to the obstacles

“r” - set the starting point for the robot

“g” - set the goal for the robot

"f" - compute the free space
 
“p” - compute the path from the start (cyan) to the goal (yellow)

“i” - initialize the program

"q" - quit
